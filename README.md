# 🛍️ Local Artisan Marketplace

A full-stack web application designed to showcase and promote handmade products by local artisans. The platform provides a clean and user-friendly interface for exploring unique handcrafted items while supporting small-scale creators.

---

## 🚀 Features

* 🧑‍🎨 Display of handmade artisan products
* 📂 Organized product categories and listings
* 🔍 Simple and intuitive browsing experience
* ⚙️ RESTful API for backend operations
* 🤖 Integrated AI-based features (Gemini / OpenRouter APIs)

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

## ⚙️ Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/Krishn-01/local-artisan-marketplace.git
cd local-artisan-marketplace
```

---

### 2️⃣ Run Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Run Frontend

```
cd frontend
npm install
npm run dev
```

> If no dev server is configured, open `index.html` directly in the browser.

---

## 🌐 Application Overview

This project demonstrates a structured full-stack architecture where the frontend and backend are developed and maintained independently. The backend handles API requests and data management, while the frontend provides an interactive interface for users.
